Sobre Mim

Sou um Desenvolvedor Full-Stack e Engenheiro de Sistemas com uma base sólida em programação e experiência prática em ambientes profissionais de alta exigência. A minha especialidade reside na criação de soluções completas, desde a interface de utilizador (Frontend) até à lógica de servidor (Backend) e integração com hardware (IoT).

A minha experiência profissional inclui um estágio na wTVision, uma referência global em soluções de broadcast para televisão e média. Durante este período, fui responsável pelo desenvolvimento de aplicações web críticas para a gestão de dados e gráficos em eventos desportivos de grande escala, como a UEFA Nations League. Esta experiência permitiu-me dominar o JavaScript e o framework Vue.js, e trabalhar com tecnologias de templating como o Scriban e formatos de dados como JSON e XML.

Em paralelo, o meu projeto de aptidão profissional (PAP) demonstrou a minha proficiência em IoT e sistemas embarcados. Desenvolvi um Robô Remoto Controlado por Voz que integra Python (Flask), WebSockets para comunicação em tempo real, MySQL e controlo de hardware (Raspberry Pi). Este projeto é a prova da minha capacidade de entregar soluções complexas que unem o mundo digital e o físico.

O que posso fazer por si:

  • Desenvolvimento Web Full-Stack: Construção de aplicações web robustas e escaláveis com JavaScript (Vue.js), Python (Flask), PHP e MySQL/SQL.
  • Sistemas IoT e Hardware: Desenvolvimento de soluções de controlo e monitorização baseadas em Raspberry Pi e Python.
  • Processamento de Dados e Integração: Criação de ferramentas para manipulação de dados (JSON/XML) e integração de APIs externas.

Procuro projetos que exijam versatilidade, rigor técnico e a capacidade de transformar ideias complexas em soluções funcionais e eficientes.

Projetos em Destaque

Robô Remoto de Vigilância e Interação (Full-Stack e IoT)

O Desafio: Conceber e construir um Robô Remoto de Vigilância e Interação de baixo custo, controlado a partir de qualquer lugar através de uma interface web intuitiva, integrando múltiplos sistemas em tempo real.

Funcionalidades Chave:

  • Controlo Remoto Full-Stack: Interface web responsiva para controlo total dos movimentos do robô (WASD).
  • Streaming de Vídeo e Captura em Tempo Real: Transmissão de imagens em tempo real e armazenamento de fotos/vídeos.
  • Controlo por Comandos de Voz: Sistema de reconhecimento de voz que traduz comandos verbais em ações do robô.
  • Monitorização de Sistema em Tempo Real: Apresentação contínua de métricas vitais da Raspberry Pi (CPU, RAM, Uptime).
  • Integração de APIs Externas: Utilização de APIs de meteorologia (OpenWeatherMap) e de rede.

Tecnologias Utilizadas:

  • Python (Flask)
  • JavaScript (WebSockets)
  • HTML5 / CSS3
  • MySQL
  • Raspberry Pi (IoT)
  • Vue.js (Implícito na experiência)

Conclusão: Este projeto demonstra proficiência em programação web (full-stack), bases de dados (SQL) e sistemas embarcados (Python/RPI), ideal para clientes que procuram desenvolvimento de IoT e soluções de controlo remoto.
